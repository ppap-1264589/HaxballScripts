# HaxballScripts
Các script của haxball mà tôi vibe code ra.

# Lưu ý
Các tool này chỉ dùng được trên máy tính, chưa có phương pháp làm trên điện thoại.

# Mô tả chức năng
| Folder | Chức năng |
|---|---|
| Direction_Avatar_and_Spam_Kick | Code để đổi avatar theo hướng đi và auto spam kick |
| Geo_Overrider | Code để đổi trường 'geo' trong localStorage của trình duyệt (đổi flag ingame) |
